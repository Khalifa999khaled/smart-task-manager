# 📝 Smart Task Manager - مدير المهام الذكي

هذا المشروع عبارة عن تطبيق بسيط لإدارة المهام اليومية، تم بناؤه باستخدام تقنيات الويب الأساسية. الهدف منه هو تقديم تجربة مستخدم سلسة مع واجهة جذابة.

## 🚀 المميزات (Features)
*   **إضافة وحذف المهام:** يمكنك تنظيم يومك بسهولة.
*   **الوضع الليلي (Dark Mode):** مريح للعين ويمكن التبديل بينه وبين الوضع الفاتح.
*   **التخزين المحلي (Local Storage):** مهامك لن تضيع حتى لو قمت بإغلاق المتصفح أو تحديث الصفحة.
*   **تصميم متجاوب:** يعمل بشكل ممتاز على الموبايل والكمبيوتر.

## 🛠️ التقنيات المستخدمة (Tech Stack)
*   **HTML5:** لبناء هيكل التطبيق.
*   **CSS3:** للتنسيق واستخدام المتغيرات (Variables) للوضع الليلي.
*   **JavaScript (Vanilla):** للتحكم في منطق التطبيق والتعامل مع الـ Local Storage.
